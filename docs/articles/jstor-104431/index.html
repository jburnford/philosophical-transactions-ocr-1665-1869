ERRATUM. Page 48. for John Machin, &c. read Martin Folkes, &c. N.B. This Number 473. begins, by Mistake of the Printer, with Page 1. Letter A. instead of Page 103. Letter O. which is the Page and Letter following the preceding Number, which begins the XLIII. Volume. Advertisement. Mr. Preston's Map of Zetland, (vulgo Shetland) mention'd p. 64. is to be sold only at Mr. Ames's near the Hermitage, Wappin.
